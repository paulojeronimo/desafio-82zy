1. Local mirror configuration (ONLY IF YOU HAVE ONE)

$ echo 'Server = http://10.0.2.2:8000/$repo/os/$arch' | \
sudo tee /etc/pacman.d/mirrorlist
$ sudo pacman -Syy

2. Required packages installation

$ sudo pacman --noconfirm -S lsb-release vim git libnghttp2
Note

The package libnghttp2 is required to fix a bug after nodejs package installation.

3. dotfiles installation

$ REPO=paulojeronimo/dotfiles && git clone https://github.com/$REPO && cd ${REPO##*/}
$ ./install
$ source ~/.bashrc

4. Node installation

$ sudo pacman --noconfirm -S nodejs
$ node -v
v12.9.1

5. Yarn installation

$ sudo pacman --noconfirm -S yarn
$ yarn -v
1.17.3
